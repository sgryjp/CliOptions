CliOptions.jl: Parsing command line options

ðŸš§UNDER HEAVY CONSTRUCTIONðŸš§

CliOptions.jl is a library for parsing command line options. There is no need to learn DSL (domain specific language) to use this library. The most basic steps to use this library is:

  1. Define specification of the command line options
  2. Parse command line arguments
  3. Optionally, merge the result with other option values (e.g.: environment variable or configuration files)
  4. Use the resolved option values
